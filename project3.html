Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects; for example:

  • Client-side JavaScript extends the core language by supplying objects relevant to controlling a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.
  • Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server. For example, server-side extensions allow an application to communicate with a database, provide continuity of information from one invocation to another, or perform file manipulations on a server.

JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in others. The JavaScript language resembles Java but does not have Java's static typing and strong type checking. JavaScript follows Java's expression syntax, naming conventions, and basic control-flow constructs, which was the reason why it was renamed from LiveScript to JavaScript.

In contrast to Java's compile-time system of classes and inheritance, JavaScript supports a runtime system based on a small number of data types representing primitive values and objects. JavaScript's object model is based on a prototype-based inheritance rather than the more common class-based inheritance.

The two languages are also very different in their typical use: Java is a general-purpose, object-oriented programming language, designed for building robust, large-scale applications. JavaScript, on the other hand, is a special-purpose scripting language designed for adding interactivity to HTML documents and is well suited for rapid application development.

Hello world

To get started with writing JavaScript, open the Scratchpad and write your first "Hello world" JavaScript code:

function greet(name) { console.log("Hello, " + name + "!"); } greet("world");

Select the code in the Scratchpad and press Ctrl+R to watch it unfold in your browser's console.

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($). Subsequent characters can also be digits (0-9). Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) and "a" through "z" (lowercase).

You can use two main types of variables: global and local. A global variable is a variable that is available throughout your code, while a local variable is only available within the function where it is declared.

var myVariable = "Hello"; let anotherVariable = 123; const fixedValue = true;
Data types

The latest ECMAScript standard defines seven data types:

  • Six data types that are primitives:
    • Boolean
    • Null
    • Undefined
    • Number
    • String
    • Symbol (new in ES6)
  • and Object

A number can be an integer or a floating-point number. A string is a sequence of characters. Boolean can be true or false.

let num = 10; // Number let str = "text"; // String let bool = false; // Boolean let obj = { key: "value" }; // Object let arr = [1, 2, 3]; // Array (which is a type of object) let n = null; // Null let u = undefined; // Undefined let sym = Symbol('description'); // Symbol
Conditional statements

Conditional statements are used to perform different actions based on different conditions. The most common conditional statements are if, else if, and else.

if (condition) { // code to be executed if condition is true } else if (anotherCondition) { // code to be executed if anotherCondition is true } else { // code to be executed if none of the above conditions are true }

You can also use the switch statement for multiple branches of execution.

switch (expression) { case value1: // code block break; case value2: // code block break; default: // code block }
Functions

A function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

function myFunction(parameter1, parameter2) { return parameter1 * parameter2; } let result = myFunction(4, 5);

Functions can be defined with or without parameters. If parameters are used, they act as local variables within the function.

Loops

Loops are used to execute the same block of code a specified number of times or while a specified condition is true.

The most common loop types are for, while, and do...while.

for (let i = 0; i < 5; i++) { console.log(i); } let i = 0; while (i < 5) { console.log(i); i++; } let j = 0; do { console.log(j); j++; } while (j < 5);
Operators

JavaScript supports the following types of operators:

  • Assignment operators
  • Comparison operators
  • Arithmetic operators
  • Bitwise operators
  • Logical operators
  • String operators
  • Conditional (ternary) operator

For example, the addition operator (+) adds two numbers. The assignment operator (=) assigns a value to a variable.

let x = 10; let y = 5; let sum = x + y; // Arithmetic operator let isEqual = (x === y); // Comparison operator let isTrue = (x > 0 && y < 10); // Logical operator
